JavaScript Array Methods: Map, Filter, and Reduce

**Map**, **Filter**, and **Reduce** are immutable array methods. This means they **do not change the original array**; instead, they always return a **new array** (or a single value, in the case of `reduce`). They are essential tools for data transformation and analysis.

---

1. The üó∫Ô∏è Map Method (Transformation)

The `map()` method is used to **transform** every element in an array and return a new array with the same number of elements.

  • **Purpose:** Change the data type or value of each item.
  • **Returns:** A **new array** of the same length.
const numbers = [1, 2, 3];

// Task: Multiply every number by 2
const doubled = numbers.map(function(num) {
  return num * 2;
});

console.log(doubled); // Output: [2, 4, 6]
console.log(numbers); // Output: [1, 2, 3] (Original array is unchanged)
---

2. The ‚úÇÔ∏è Filter Method (Selection)

The `filter()` method is used to **select** a subset of elements from an array based on a specific condition.

  • **Purpose:** Keep only the items that pass a boolean test.
  • **Returns:** A **new array** that may be shorter than the original. 
const ages = [15, 20, 25, 17, 30];

// Task: Keep only ages 18 and older
const adults = ages.filter(function(age) {
  return age >= 18; // Must return true or false
});

console.log(adults); // Output: [20, 25, 30]
---

3. The üìâ Reduce Method (Aggregation)

The `reduce()` method is used to **aggregate** all elements in an array into a single output value (e.g., a number, a string, or an object). It is the most versatile of the three.

  • **Purpose:** Calculate a single value from an array (like a sum or average).
  • **Returns:** A **single value**.
const items = [10, 20, 30, 40];

// Task: Calculate the total sum of all elements
const sum = items.reduce(function(accumulator, currentValue) {
  // accumulator is the running total from the previous iteration
  return accumulator + currentValue;
}, 0); // 0 is the initial value for the accumulator

console.log(sum); // Output: 100
